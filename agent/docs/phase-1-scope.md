# Faz 1 kapsamı

Bu faz yalnızca Windows Desktop Agent MVP'sidir: WPF ekranları, grup/şablon yönetimi, açık onaylı sıralı toplu gönderim, iptal, geçmiş, retry iş kuralı, SQLite, log ve artifact yönetimi.

Backend, API, PostgreSQL, React, Docker, n8n, AI, gelen mesaj analizi, anket, zamanlama ve bulut entegrasyonu kapsam dışıdır.

POC geçiş tamamlanana ve gerçek olmayan test gruplarıyla manuel checklist geçene kadar korunacaktır. Otomatik gerçek WhatsApp entegrasyon testi unit test paketine dahil edilmez.

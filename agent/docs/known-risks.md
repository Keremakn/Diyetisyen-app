# Bilinen riskler

- WhatsApp Web DOM/aria/data-testid değişiklikleri selector'ları bozabilir.
- WhatsApp otomasyonu resmi API değildir; hesap veya kullanım politikası riski ayrıca değerlendirilmelidir.
- CAPTCHA, oturum düşmesi ve genel ağ kesintisi güvenli duruş üretir; manuel müdahale gerekebilir.
- Tik doğrulaması WhatsApp UI durumuna bağlıdır; uzun ağ gecikmeleri zaman aşımı yaratabilir.
- Faz 0'daki kırmızı hata menüsünden yeniden gönderme varyantları yeni gateway'e henüz taşınmadı; Faz 1 varsayılan öğe denemesi 1'dir.
- Şablon ve grup düzenleme/pasifleştirme servis sınırında hazırdır; MVP UI'daki ayrıntılı düzenleme deneyimi iyileştirilebilir.
- Uygulama kapanırken Playwright/SQLite güvenli kapatılır; işletim sistemi tarafından zorla sonlandırmada Processing batch yeniden açıldığında geçmişte görünür, otomatik devam etmez.
